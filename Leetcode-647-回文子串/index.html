Leetcode 647.回文子串

题目

给定一个字符串，你的任务是计算这个字符串中有多少个回文子串。

具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被计为是不同的子串。

示例2：

注意：

思路 + 代码

回文子串的内部一定是回文子串，因此关键在于重复利用回文子串已经统计过的数值。

可采用双指针由回文子串向两端同时检测的方法。

分为偶数回文子串与奇数回文子串。

思路 + 代码

采用动态规划，用一个二维数组 dp[i][j] 统计字符串从 i ~ j 是否是回文子串。
