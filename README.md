# hexo 使用方法

1. `hexo clean` -- 清除文件

2. `hexo algolia` -- 生成 algolia 搜索索引

3. `hexo douban -b -m` -- 生成豆瓣电影及书单 (需要 node 14 版本)

4. `hexo g` -- 生成博客文件

5. `hexo s` -- 启动 hexo 博客

6. `hexo deploy` -- 部署 hexo 博客

7. `hexo n 文件名` -- 新建 hexo 博客文章
